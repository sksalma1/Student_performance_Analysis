# Education Analytics for Student Performance

This project focuses on analyzing student academic performance using a data-driven approach. The goal is to understand how different factors such as course structure, assessments, student activity in the Virtual Learning Environment (VLE), and registration details influence student outcomes.

The system uses a data warehouse architecture where multiple datasets are processed, transformed, and integrated into dimensional tables and fact tables using Python and PostgreSQL. The structured data is then used to perform analytical queries and generate insights about student behavior and performance patterns.

An interactive Power BI dashboard is developed on top of the data warehouse to visualize key metrics such as student engagement, assessment performance, course-wise outcomes, and risk indicators for low-performing students. These visualizations help educators and administrators make informed decisions to improve academic performance and student support strategies.

## Key Features:

* Data preprocessing and ETL pipeline implemented using Python
* Data warehouse design with dimension and fact tables in PostgreSQL
* SQL queries for analytical insights on student performance
* Interactive Power BI dashboard for visualization and reporting
* Identification of factors affecting student success and engagement

## Technologies Used:
Python, PostgreSQL, SQL, Power BI, Jupyter Notebook

## Dataset:
The project uses a student learning analytics dataset containing information about courses, assessments, student registrations, and interactions with the virtual learning environment.

## Outcome:
The project demonstrates how data analytics and visualization techniques can be used in the education sector to monitor student progress, identify patterns, and support data-driven decision making in academic institutions.
